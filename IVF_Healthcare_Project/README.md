# IVF Healthcare Analysis Project

## Project Overview
This project focuses on analyzing IVF healthcare and fertility treatment data to identify treatment success trends, patient patterns, and 
operational insights. The analysis was performed using Python, SQL, Excel, Power BI, and Tableau to support data-driven decision-making in 
the healthcare domain.

## Business Problem
Fertility centers often face challenges in tracking IVF treatment outcomes, patient response patterns, and operational performance due to 
scattered and unstructured data. The objective of this project is to preprocess, analyze, and visualize IVF treatment data for better clinical 
and business insights.

## Tools & Technologies Used
- Python
- SQL
- Power BI
- Tableau
- Excel
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Tasks Performed
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Missing Value Handling
- Outlier Detection & Treatment
- KPI Development
- Dashboard Creation
- Patient Trend Analysis
- Business Insight Generation

## Project Features
- IVF success rate analysis
- Hormone level and treatment trend analysis
- Embryo transfer and oocyte analysis
- Patient demographic analysis
- Statistical summaries and distributions
- Correlation and multivariate analysis
- Interactive healthcare dashboards

## Business Insights
- Identified factors influencing IVF treatment outcomes
- Analyzed patient and treatment success patterns
- Improved visibility into operational and clinical KPIs
- Supported healthcare reporting and performance monitoring

## Outcome
The project helps fertility centers improve reporting accuracy, monitor treatment performance, analyze patient trends, and support data-driven
healthcare decisions through interactive dashboards and analytical insights.
