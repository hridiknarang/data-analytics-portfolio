# Power Trading Analysis Project

## Project Overview
This project focuses on analyzing power trading and electricity market data to identify pricing trends, demand-supply patterns, and market behavior. The analysis was performed using Python, SQL, Excel, Power BI, and Tableau to generate business insights and interactive dashboards.

## Business Problem
Electricity prices are highly volatile due to demand fluctuations, weather conditions, and market uncertainties. The objective of this project is to preprocess and analyze trading data to support better operational and trading decisions.

## Tools & Technologies Used
- Python
- SQL
- Power BI
- Tableau
- Excel
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Tasks Performed
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Outlier Detection & Handling
- KPI Development
- Dashboard Creation
- Multivariate Analysis
- Business Insight Generation

## Project Features
- MCP and MCV trend analysis
- Demand and supply behavior analysis
- Weather impact analysis
- Statistical summaries and distributions
- Correlation and multivariate visualizations
- Interactive business dashboards

## Business Insights
- Identified market price volatility trends
- Analyzed relationships between weather conditions and electricity demand
- Detected abnormal trading patterns and outliers
- Improved visibility into trading performance and operational efficiency

## Outcome
The project helps in understanding electricity market behavior, optimizing trading decisions, improving reporting accuracy, and supporting data-driven business strategies.
